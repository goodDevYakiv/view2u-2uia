Про проєкт

View2U — це автономна система інтелектуального збору даних з інтеграцією ШІ.

🛠️Вбудовані інструменти

  1. Форми - гнучкий збір даних з будь-яких полів.
  2. Геолокація - отримання точних координат пристрою.
  3. Фото - збір зображень з фронтальної та основної камери.
  4. Відео - запис відео з обох камер.
  5. Автоматичний збір - IP, UserAgent, мова, екран, часова зона, платформа та ін.

🚀Версія 0.2.0 (2025-09-17)

  • Інтерфейс: Повністю перероблено для кращої взаємодії.
  • Відео: Додано функціонал запису відео.
  • Telegram Бот (v1.1.0): Покращено стабільність та підтримку нових типів даних.
  • Генерація ШІ (v0.2.0): Покращено якість генерації сторінок.
  • Стабільність: Виправлено критичні помилки системи.

🔮Плани на майбутнє

  • Інструмент для отримання сесії Telegram.
  • Інструмент запису аудіо.
  • Виправлення помилки з відображенням завантажувача під модальними вікнами.
  • Можливість створення тимчасових акаунтів (для Адмін-панелі).
